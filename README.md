# Simple-Crypto-Agent-Tool
A Python-based tool to fetch real-time crypto prices and provide basic market analysis. A foundation for building autonomous AI Agents on Bitcoin.

# 本项目是为旨在展示如何使用 Python 构建 AI Agent 的基础组件：**实时链外数据获取 (Oracle-like data fetching)**。

## 功能 (Current Features)
- 实时追踪 BTC/ETH 价格。
- 简单的自动决策逻辑模拟 (Basic Decision Logic)。
- 为未来接入 GOAT Network 的 x402 协议做准备。

在 Agent Economy 中，智能体需要感知市场价格才能进行自动支付或资产转换。本项目通过 Python 展示了 Agent 获取外部信息的能力。

## 如何运行 (How to run)
1. `pip install requests`
2. `python agent_price_tool.py`
